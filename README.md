# Brain Tumor Detection using Machine Learning

This project detects brain tumors from MRI images using machine learning and image processing techniques.

## Technologies
- Python
- Scikit-learn
- NumPy
- OpenCV
- Matplotlib
- Seaborn

## Methods
- Image preprocessing
- Feature extraction
  - GLCM texture features
  - Edge detection (Sobel)
  - Histogram features
- Machine Learning models
  - SVM
  - Random Forest
  - Gradient Boosting
  - KNN
  - Logistic Regression

## Dataset
MRI images categorized into:
- Tumor (yes)
- No tumor (no)

## Evaluation Metrics
- Accuracy
- ROC-AUC
- Confusion Matrix
- Classification Report
